# Lab Setup 
The username for SSH is    
`ec2-user`

## MacOS 
Download `lab.pem` from the `keys` directory

### Set permission on SSH key 
```
chmod 600 /path/to/lab.pem
```

### SSH to lab servers 
```
ssh -i /path/to/lab.pem <user>@<LAB IP> 
```
